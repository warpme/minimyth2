Overview

Minimyth2 is a small Linux distribution that turns a diskless computer into a MythTV frontend.

This version of MiniMyth2 is version @MM_VERSION_MINIMYTH@ and contains:

  • MythTV build from sources through @MYTHTV_SVN_VERSION@ with

    MythTV Plugins:

    • MythBrowser
    • MythGallery
    • MythGame(not integrated with Minimyth configuration)
    • MythMusic
    • MythNetvision
    • MythNews
    • MythWeather
    • MythZoneMinder

    MythTV build-in Themes:

    • Monochrome

  • Major System Components:

    • Linux @LINUX_FULL_VERSION@
    • Weston 13.0.3
    • Xorg server 21.1.13
    • LIRC 0.10.2
    • ALSA 1.2.12
    • MESA 24.1.4
    • Qt 5.15.14
    • Qt WebKit 5.212 g84ad475fca
    • Intel Wireless Daemon (iwd) 2.19

  • Major Build Tools/Libraries:

    • GNU Binutils 2.40
    • GCC compiler 14.2.0
    • LLVM compiler 18.1.8
    • GNU C Library 2.39/master gfd658f026

  • Key Video Drivers:

    • Intel VAAPI driver g3ed3f6a783fdf
    • Intel media driver 24.2.2
    • VAAPI libva library 2.20.0
    • DRM libdrm library 2.4.122
