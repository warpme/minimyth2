Overview

Minimyth2 is a small Linux distribution that turns a diskless computer into a MythTV frontend.

This version of MiniMyth2 is version @MM_VERSION_MINIMYTH@ and contains:

  • MythTV build from sources through @MYTHTV_SVN_VERSION@ with

    MythTV Plugins:

    • MythBrowser
    • MythGallery
    • MythGame(not integrated with Minimyth configuration)
    • MythMusic
    • MythNetvision
    • MythNews
    • MythWeather
    • MythZoneMinder

    MythTV build-in Themes:

    • Monochrome

    Kodi:

    • Kodi Piers (v22) gcb75fb5fc36

  • Major System Components:

    • Linux @LINUX_FULL_VERSION@
    • Weston 13.0.3
    • Xorg server 21.1.18
    • LIRC 0.10.2
    • ALSA 1.2.13
    • MESA 25.1.6
    • Qt 5.15.17
    • Intel Wireless Daemon (iwd) 3.9

  • Major Build Tools/Libraries:

    • GNU Binutils 2.44
    • GCC compiler 15.1.0
    • LLVM compiler 20.1.4
    • GNU C Library 2.41/master 20250228-g89be78704

  • Key Video Drivers:

    • Intel VAAPI driver gfd727a4e9c
    • Intel media driver 25.1.4
    • VAAPI libva library 2.22.0
    • DRM libdrm library 2.4.125
